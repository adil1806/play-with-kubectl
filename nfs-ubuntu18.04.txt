



::::Ubuntu 18.04 NFS:::

::: from nfs server machine :::
1. Update package: 
sudo apt-get update

2. Install nfs-kernel-server
sudo apt-get install nfs-kernel-server -y

3. Enable nfs server
systemctl enable nfs-kernel-server

4. Start nfs server
systemctl start nfs-kernel-server

5. Now, create a directory
mkdir /nfs-private

6. Give permission
sudo mkdir /nfs-private

7. Now, add this nfs directory in /etc/exports
sudo vim /etc/exports
add this line at bottom of this file:

/nfs-private  *(rw,sync,no_subtree_check)

8. Now export this
sudo exportfs -arvf
root@master:/nfs-private# sudo exportfs -arvf
exporting *:/nfs-private

9. Now, restart nfs server
sudo systemctl restart nfs-kernel-server



::: from nfs client machine :::
::: Temp :::

1. Update package: 
sudo apt-get update

2. Install nfs-common
sudo apt-get install nfs-common -y

3. Check nfs file from client to server
sudo showmount -e 3.20.66.155

4. create a directory in client to mount nfs
mkdir /nfs-client

5. M0unt 
sudo mount -t nfs 3.20.66.155:/nfs-private /nfs-client

6. Check mount
mount | grep nfs

3.20.66.155:/nfs-private on /nfs-client type nfs4 (rw,relatime,vers=4.2,rsize=524288,wsize=524288,namlen=255,hard,proto=tcp,timeo=600,retrans=2,sec=sys,clientaddr=172.31.32.180,local_lock=none,addr=3.20.66.155)

7. 
ubuntu@ip-172-31-32-180:~$ df -h
